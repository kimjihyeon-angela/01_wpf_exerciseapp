- 주제 : 운동기록, 운동 성장 일지??

- 현황 및 방향
  1) 현재 운동 예약 프로그램(어플)이 존재함(ex - 피플박스)
      피플박스는 정말 운동 예약을 위한 프로그램임
      -> 나의 기록을 입력, 저장하려면 핸드폰에서 메모장에 저장하거나 사진을 찍어야함
         => 내 기록을 입력, 저장할 수 있는 공간 만들기 
         => 기록들을 한번에 볼 수 있는 표 제공
         => 같은 운동에 대한 성장 그래프 제공 
         => 기록을 입력하지 않는 회원들을 위해 대표자, 코치 등 관리자가 직접 입력하는 기능도 넣기

  2) 피플박스 오늘의 운동 
      -> 날짜를 하나하나 눌러 확인해야함
         => 달력을 제공하여 전체적인 운동 내역을 볼 수 있도록 제공
         => 달력의 날짜를 선택하면 그날의 운동을 자세히 볼 수 있도록 제공
         => 오늘의 운동이 끝난 밤 12시 같이 운동하는 사람들의 평균치를 달력에 작성하여 입력해주기

  3) 대회방만들기
      -> 대회를 열 경우 한 사람이 엑셀로 정리하여 사람들에게 제공해 줄 때까지 결과를 알 수 없음
         => 대회방을 만들어 참여자들을 초대할 경우 실시간으로 순위 및 결과 알려줌
         => 대회방 만들때 대회명, 룰북 첨부 가능하게 만들기

  4) 운동 설명
     -> 오늘의 운동이 올라왔을 때 모르는 운동은 직접 찾아봐야 함
         => 유튜브 링크 제공 or 동영상 제공하여 미리 볼 수 있게 만들기

  5) 타이머 제공
         => emom, tabata, fortime, countdown 크로스핏에서 사용하는 타이머 제공 

  6) 무게 계산
      -> 무게는 항상 본인들이 직접 계산해서 운동했음
         => 관리자가 운동 무게 퍼센트를 지정해주면 본인들의 저장되어 있는 기록에서 자동 계산되어 무게 알려줌
	ex) 운동 : 백스쿼트 75% 5개
	     나에게 보이는 화면 : 백스쿼트 75%(110Lb) 5개

  7) 커뮤니티..??
         => 같은 박스 내의 사람들과의 커뮤니티 공간
         => 전체 공간

  8) 관리자 입장에서 운동입력하기 편하게 만들기 
         => 히어로 와드 드롭박스 형태로 넣어놓기 
	(히어로 와드 이름 드롭박스로 넣어놓으면 어떤 조합인지 나오게)
         => 스쿼트, 스내치, 클린 등 주요 동작들도 넣어놓기
         => Rxd, Scale 무게 설정 따로 가능하게 해놓기
         => 사용자 입장에서는 내가 Rxd로 운동했는지 Scale로 운동했는지 체크할 수 있는 체크박스 만들기

- 효과
  1) 나의 운동 성장 그래프를 눈으로 직접 확인해 볼 수 있음 
  2) 어떤 운동을 했는지 전체적으로 볼 수 있게 됨 (달력, Rxd & Scale 선택 체크박스)
  3) 같은 운동하는 사람들과 더 많은 이야기로 소통할 수 있음
  4) 대회를 열 경우 나의 순위를 실시간으로 확인해 볼 수 있게 됨
